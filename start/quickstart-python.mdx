Follow one path to a visible result using the official Python SDK. Prefer JavaScript? See [Quickstart (JavaScript)](/start/quickstart-js).

Prerequisites:
- Python 3.8 or higher
- Valid Moss project credentials (`MOSS_PROJECT_ID`, `MOSS_PROJECT_KEY`)
- `inferedge-moss` and `.env` to load credentials

## 1) Install

```bash
python -m venv .venv && source .venv/bin/activate
pip install inferedge-moss python-dotenv
```

## 2) Configure credentials

From the Moss portal, copy your project credentials. Choose one:

**A) Export in shell**

```bash
export MOSS_PROJECT_ID="your_project_id"
export MOSS_PROJECT_KEY="your_project_key"
```

**B) Use a `.env` file** (recommended for local dev)

```bash
echo "MOSS_PROJECT_ID=your_project_id" >> .env
echo "MOSS_PROJECT_KEY=your_project_key" >> .env
```

## 3) Create and query an index

Save as `quickstart.py` and run with `python quickstart.py` (Python 3.9+):

```python
import os
import asyncio
from dotenv import load_dotenv
from inferedge_moss import MossClient, DocumentInfo

load_dotenv()  # loads .env
PROJECT_ID = os.getenv("MOSS_PROJECT_ID")
PROJECT_KEY = os.getenv("MOSS_PROJECT_KEY")

async def main():
    client = MossClient(PROJECT_ID, PROJECT_KEY)
    index_name = f"quickstart-{os.getpid()}"

    documents = [
        DocumentInfo(
            id="doc1",
            text="How do I track my order? You can track your order by logging into your account.",
            metadata={"category": "shipping"}
        ),
        DocumentInfo(
            id="doc2",
            text="What is your return policy? We offer a 30-day return policy for most items.",
            metadata={"category": "returns"}
        ),
        DocumentInfo(
            id="doc3",
            text="How can I change my shipping address? Contact our customer service team.",
            metadata={"category": "support"}
        )
    ]

    await client.create_index(index_name, documents, "moss-minilm")  # default; use "moss-mediumlm" for higher accuracy

    await client.load_index(index_name)
    results = await client.query(index_name, "How do I return a damaged product?", top_k=3, alpha=0.6)
    print(results.docs[0])

asyncio.run(main())
```

### Example output

```json
{
  "id": "doc2",
  "score": 0.88,
  "text": "What is your return policy? We offer a 30-day return policy for most items."
}
```

<Tip>Need keyword-heavy results? Use `alpha=0.0` for pure keyword, `alpha=1.0` for pure semantic, or a blend in between (see Retrieval for details).</Tip>

## Load an existing index (optional)

Already have an index? Set `MOSS_INDEX_NAME` in `.env` and query it directly (mirrors `moss-samples/python/load_and_query_sample.py`):

```python
import os, asyncio
from dotenv import load_dotenv
from inferedge_moss import MossClient

load_dotenv()

async def main():
    client = MossClient(os.getenv("MOSS_PROJECT_ID"), os.getenv("MOSS_PROJECT_KEY"))
    index = os.getenv("MOSS_INDEX_NAME")
    await client.load_index(index)
    results = await client.query(index, "refund processing time and policy", 3)
    print([{ "id": d.id, "score": round(d.score, 3) } for d in results.docs])

asyncio.run(main())
```
