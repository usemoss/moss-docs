Follow one path to a visible result using the official JS SDK. Prefer Python? See [Quickstart (Python)](/start/quickstart-python).

Prerequisites:
- Node.js 16+
- Valid Moss project credentials (`MOSS_PROJECT_ID`, `MOSS_PROJECT_KEY`)
- `@inferedge/moss` and `.env` to load credentials

## 1) Install

```bash
npm install @inferedge/moss dotenv
```

## 2) Configure credentials

From the Moss portal, copy your project credentials. Choose one:

**A) Export in shell**

```bash
export MOSS_PROJECT_ID="your_project_id"
export MOSS_PROJECT_KEY="your_project_key"
```

**B) Use a `.env` file** (recommended for local dev)

```bash
echo "MOSS_PROJECT_ID=your_project_id" >> .env
echo "MOSS_PROJECT_KEY=your_project_key" >> .env
```

## 3) Create and query an index

Save as `quickstart.ts` and run with `npx tsx quickstart.ts` (Node 18+):

```ts
import { MossClient, DocumentInfo } from '@inferedge/moss'
import { config } from 'dotenv'
config() // loads .env

const projectId = process.env.MOSS_PROJECT_ID!
const projectKey = process.env.MOSS_PROJECT_KEY!

async function main() {
  // Initialize search client with project credentials
  const client = new MossClient(projectId, projectKey)

  // Prepare documents to index
  const documents: DocumentInfo[] = [
    {
      id: 'doc1',
      text: 'How do I track my order? You can track your order by logging into your account.',
      metadata: { category: 'shipping' },
    },
    {
      id: 'doc2',
      text: 'What is your return policy? We offer a 30-day return policy for most items.',
      metadata: { category: 'returns' },
    },
    {
      id: 'doc3',
      text: 'How can I change my shipping address? Contact our customer service team.',
      metadata: { category: 'support' },
    },
  ]

  // Create an index with documents and model
  const indexName = 'faqs'
  const created = await client.createIndex(indexName, documents, 'moss-minilm') // default; use 'moss-mediumlm' for higher accuracy
  console.log('Index created:', created)

  // Load the index before searching
  await client.loadIndex(indexName)

  // Search the index
  const results = await client.query(indexName, 'How do I return a damaged product?', 3)

  // Display one result
  console.log(results.docs[0])
}

main().catch(console.error)
```

### Example output

```json
{
  "id": "doc2",
  "score": 0.88,
  "text": "What is your return policy? We offer a 30-day return policy for most items."
}
```

<Tip>Need keyword-heavy results? Use `alpha: 0.0` for pure keyword, `alpha: 1.0` for pure semantic, or a blend in between (see Retrieval for details).</Tip>

## Load an existing index (optional)

Already have an index? Set `MOSS_INDEX_NAME` in `.env` and query it directly (mirrors `moss-samples/javascript/load_and_query_sample.ts`):

```ts
import { MossClient } from '@inferedge/moss'
import { config } from 'dotenv'
config()

const client = new MossClient(process.env.MOSS_PROJECT_ID!, process.env.MOSS_PROJECT_KEY!)

async function main() {
  await client.loadIndex(process.env.MOSS_INDEX_NAME!)
  const results = await client.query(process.env.MOSS_INDEX_NAME!, 'refund processing time and policy', 3)
  console.log(results.docs.map(d => ({ id: d.id, score: d.score.toFixed(3) })))
}

main().catch(console.error)
```
