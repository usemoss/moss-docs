Follow one path to a visible result using the official JS SDK.

## 1) Install

```bash
npm install @inferedge/moss dotenv
```

## 2) Configure credentials

From the Moss portal, copy your project credentials and set env vars:

```bash
export MOSS_PROJECT_ID=your_project_id
export MOSS_PROJECT_KEY=your_project_key
```

## 3) Create and query an index

Save as `quickstart.ts` and run with `npx tsx quickstart.ts` (Node 18+):

```ts
import { MossClient } from '@inferedge/moss'
import { config } from 'dotenv'
config()

const projectId = process.env.MOSS_PROJECT_ID!
const projectKey = process.env.MOSS_PROJECT_KEY!

async function main() {
  const client = new MossClient(projectId, projectKey)
  const indexName = `quickstart-${Date.now()}`

  await client.createIndex(indexName, [
    { id: '1', text: 'Moss supports offline-first semantic search.' },
    { id: '2', text: 'Queries can complete in sub-10ms.' },
    { id: '3', text: 'Add long-term memory to AI agents.' },
  ], 'moss-minilm')

  await client.loadIndex(indexName)
  const results = await client.query(indexName, 'How fast are local queries?', 3)
  console.log(results.docs[0])
}

main().catch(console.error)
```

### Example output

```json
{
  "id": "2",
  "score": 0.92,
  "text": "Queries can complete in sub-10ms."
}
```
