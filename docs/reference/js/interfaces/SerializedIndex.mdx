# Interface: SerializedIndex

Complete serialized representation of an index for local .moss file storage.

Contains all data necessary to recreate an index, including configuration,
documents, and pre-computed embeddings.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| `name` | `string` | Name of the index. |
| `version` | `string` | Index build/format version (semver). |
| `model` | [`ModelRef`](./ModelRef) | Model bound to this index. |
| `dimension` | `number` | Embedding dimensionality. |
| `embeddings` | `number`[][] | Embedding vectors (rows = embedding vectors). |
| `docIds` | `string`[] | Document IDs parallel to embeddings. |
