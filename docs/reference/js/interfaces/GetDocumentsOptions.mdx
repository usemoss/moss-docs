# Interface: GetDocumentsOptions

Options for retrieving documents from an index.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| `docIds?` | `string`[] | Optional array of document IDs to retrieve. If omitted, returns all documents. |
