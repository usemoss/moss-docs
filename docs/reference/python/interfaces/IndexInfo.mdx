# Interface: IndexInfo

Metadata describing a Moss index, such as document counts and model bindings.

## Properties

- **id**: `str`
- **name**: `str`
- **version**: `str`
- **status**: `str`
- **doc_count**: `int`
- **created_at**: `str`
- **updated_at**: `str`
- **model**: [`ModelRef`](./ModelRef)
