## Problem
Users need precise answers from large documentation sets.

## Architecture
- Chunk docs -> embed -> index
- Query: retrieve top-k -> rerank -> answer

## Code walkthrough

```ts
import { Moss } from '@moss-ai/sdk'
const moss = new Moss()

await moss.index.upsert(chunks)
const results = await moss.search.query('reset password limits')
```

## Outcome
Accurate, fast answers in-product without leaving the page.
