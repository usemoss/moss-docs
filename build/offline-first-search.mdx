## Problem
Search must work without network access and stay private.

## Architecture
- Local embeddings and index stored on device
- Optional background sync to cloud
- Query pipeline: text -> embed -> retrieve -> rerank (optional)

## Code walkthrough (JavaScript)

```ts
import { MossClient } from '@inferedge/moss'

const client = new MossClient(process.env.MOSS_PROJECT_ID!, process.env.MOSS_PROJECT_KEY!)
const indexName = 'offline-demo'

await client.createIndex(indexName, docs, 'moss-minilm')
await client.loadIndex(indexName)
const hits = await client.query(indexName, 'install steps offline', 5)
```

## Result
Sub-10ms queries with zero network dependency.
