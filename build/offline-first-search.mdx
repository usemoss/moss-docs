## Problem
Search must work without network access and stay private.

## Architecture
- Local embeddings and index stored on device
- Optional background sync to cloud
- Query pipeline: text -> embed -> retrieve -> rerank (optional)

## Code walkthrough

```ts
import { Moss } from '@moss-ai/sdk'
const moss = new Moss()

await moss.index.upsert(docs) // local index
const hits = await moss.search.query('install steps offline')
```

## Result
Sub-10ms queries with zero network dependency.
