High-level overview of the official SDKs. See the API Reference for endpoints.

## JavaScript (`@inferedge/moss`)

```ts
import { MossClient } from '@inferedge/moss'

const client = new MossClient(projectId, projectKey)

await client.createIndex(name, docs, 'moss-minilm')
await client.loadIndex(name)
const results = await client.query(name, 'query text', 5)
await client.deleteIndex(name)
```

Types you’ll see:
- `DocumentInfo` – `{ id: string; text: string; metadata?: Record<string,string> }`
- `AddDocumentsOptions` – `{ upsert?: boolean }`

## Python (`inferedge-moss`)

```python
from inferedge_moss import MossClient

client = MossClient(project_id, project_key)
await client.create_index(name, docs, 'moss-minilm')
await client.load_index(name)
res = await client.query(name, 'query text', 5)
await client.delete_index(name)
```
