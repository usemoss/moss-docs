This section summarizes key SDK surfaces. See the API Reference for endpoints.

## Client
```
new Moss(options)
```

Options:
- `apiKey` – string (optional for local-only)
- `embeddings.provider` – `local` | `remote`

## Index
- `index.upsert(docs)` – add or update documents
- `index.delete(ids)` – remove documents

## Search
- `search.query(text, options)` – retrieve documents
